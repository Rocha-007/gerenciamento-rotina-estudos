# Instruções Rápidas (Português)

## 🚀 EXECUTE EM 30 SEGUNDOS

### Requisito único: Java 11+ instalado

### Windows ✅
1. Clone o repositório
2. Entre na pasta
3. **Clique 2x em `run.bat`**
4. Pronto! A aplicação vai abrir

### Linux/Mac ✅
1. Clone o repositório
2. Entre na pasta
3. Execute: `./run.sh`
4. Pronto! A aplicação vai abrir

---

## 📥 Passo a Passo Completo

```bash
# Clone o repositório
git clone https://github.com/Rocha-007/gerenciamento-rotina-estudos.git

# Entre na pasta
cd gerenciamento-rotina-estudos

# Windows: duplo clique em run.bat
# OU

# Linux/Mac:
chmod +x run.sh
./run.sh
```

---

## ❓ O que fazer se não tiver Java?

1. Baixe Java 11 LTS de: https://www.oracle.com/java/technologies/downloads/
2. Instale seguindo o instalador
3. Reinicie o computador
4. Execute os scripts novamente
